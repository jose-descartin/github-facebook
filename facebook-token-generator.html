Create Facebook App

Let's create a new Facebook app in the Developer Console.

Follow these steps:
  • Go to Facebook Developers ‚Üí
  • Click "Create App"
  • Type your desired App name
  • Click "Next"
2

Configure App Settings

Select the correct use case and permissions.

In the Facebook Developer Console:
  • Select "Content management"
  • Select "Manage everything on your Page"
  • Click "Next"
3

Create Portfolio & Add Page

Create a portfolio and link your Facebook page.

Follow these steps:
  • Create a new Portfolio
  • Add your Facebook page to the portfolio
  • Click "Add Page" to confirm
  • Select the created portfolio
  • Click "Next" to finish app creation
  • Click "Go to dashboard"
4

Configure Permissions

Set up the required API permissions.

In the Developer Console:
  • Click "Use cases"
  • Click "Customize"
  • Select: pages_manage_posts, pages_read_engagement, pages_show_list
  • Click "Tools" ‚Üí "Graph API Explorer"
  • Select your app from the dropdown
  • Click "Add a Permission" ‚Üí "Events Groups Pages"
  • Select all the permissions that appear
5

Generate User Access Token

Generate your first access token.

In Graph API Explorer:
  • Click "Generate Access Token"
  • Click "Continue as [Your Name]"
  • Select the page(s) where content will be posted
  • Select your created portfolio
  • Click "Save"
  • Click "Got it"
  • Copy the generated user access token
6

Publish App & Get Credentials

Publish your app and retrieve credentials.

Follow these steps:
  • In Dashboard, click "Publish"
  • Fill out required sections and create necessary files
  • Click "Save Changes" then "Publish"
  • Copy your App ID and App Secret (click "Show" to view)
7

Configure OAuth Settings

Add redirect URIs for authentication.

In the Developer Console:
  • Click "Facebook Login for Business" ‚Üí "Settings"
  • Add Valid OAuth Redirect URIs
  • Click "Save Changes"
Typical redirect URIs: https://yourdomain.com/callback or http://localhost:3000/callback for testing
8

Generate Page Access Token

Convert your user token to a page token.

Follow these steps:
  • Go back to Graph API Explorer ‚Üí
  • Click "Page Access Token" dropdown
  • Select your Facebook page name
  • A page access token will be generated
  • Copy this token
9

Generate Long-Lived Token

Convert to a long-lived token that never expires.

Final steps:
  • Go to Access Token Debugger ‚Üí
  • Paste your page token and click "Debug"
  • Click "Extend Access Token"
  • Copy the new long-lived token that appears below
  • Click "Debug" again to verify it shows "Expires: Never"
